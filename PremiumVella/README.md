# 🛍️ Smart Shopping Store

A complete e-commerce shopping cart application built with React, Tailwind CSS, and Vite.

## ✨ Features

### Core Features
- ✅ **Display 20+ Products** - Rich product catalog with images
- ✅ **Product Search** - Real-time search filtering
- ✅ **Category Filtering** - Filter by product categories
- ✅ **Add to Cart** - Add products to shopping cart
- ✅ **Remove from Cart** - Remove items from cart
- ✅ **Quantity Control** - Increase/decrease quantity
- ✅ **Dynamic Total** - Auto-calculates total price
- ✅ **Cart Counter** - Shows number of items in cart
- ✅ **Checkout Form** - Validated checkout form
- ✅ **Local Storage** - Cart and wishlist persistence
- ✅ **Responsive Design** - Works on all devices

### Bonus Features
- ✅ **Wishlist System** - Save favorite products
- ✅ **Coupon Discount** - Apply discount codes (SAVE10, SAVE20, WELCOME, SUMMER)
- ✅ **Quick View Modal** - View product details quickly

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/smart-shopping-store.git

# Navigate to project
cd smart-shopping-store

# Install dependencies
npm install

# Start development server
npm run dev
