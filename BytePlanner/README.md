# BytePlanner

> Smart Task Management System by ByteHex Tech

A professional, feature-rich task management application built with React 19, Tailwind CSS V4, and Vite.

## ✨ Features

- ✅ **Add, Edit, Delete Tasks** - Full CRUD operations
- ✅ **Mark Tasks as Completed** - Toggle completion status
- ✅ **Search Tasks** - Real-time search filtering
- ✅ **Filter Tasks** - All, Active, Completed views
- ✅ **Priority Levels** - High, Medium, Low with color coding
- ✅ **Due Dates** - Set and track deadlines
- ✅ **Local Storage** - Automatic data persistence
- ✅ **Statistics Dashboard** - Real-time analytics
- ✅ **Drag & Drop** - Intuitive task reordering
- ✅ **Dark/Light Mode** - Theme toggle with persistence
- ✅ **Responsive Design** - Works on all devices

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/byteplanner.git

# Navigate to project
cd byteplanner

# Install dependencies
npm install

# Start development server
npm run dev
